# -本项目用来提交2026 北京青少年人工智能应用实践活动

作者：刘锦泽
学校：北京市顺义区第一中学

*由于本项目使用的是本地部署的Qwen大模型，所以运行需预先下载ollama然后在ollama里面下载qwen2.5:3b模型
*需要提前下载npm库


## 作品简介

"智慧学伴"是一款基于本地大模型运行的AI智能体。它能像私人学习参谋一样，**感知**学习需求、**自主规划**分析任务、**调用工具**完成推理、**输出可视化结果**。

核心任务：接收学习参数或试卷数据 → 自主分析推理 → 生成学习计划或诊断报告。

---

 核心功能演示

 功能一：智能学习计划生成器

输入学科、当前成绩、目标分数、可用天数，智能体自主规划每日学习任务。


### 功能二：试卷智能分析器

拖拽上传 .txt 格式试卷，自动识别题型、定位薄弱知识点、生成雷达图。

---

## 技术架构

| 技术层 | 实现方案 |
|---|---|
| 前端交互 | HTML + CSS + JavaScript，单页应用，浏览器打开即用 |
| AI引擎 | Ollama 本地部署 Qwen2.5-3B 大语言模型 |
| 桥接服务 | Node.js 轻量服务，连接前端与本地模型 |
| 隐私保障 | 全本地离线部署，数据不离开本机 |

---

## 智能体工作流（试卷分析）

1. **感知**：接收 .txt 试卷文件
2. **规划**：拆解任务——扫描题型 → 定位错题 → 映射知识点标签
3. **调用工具**：调用 Qwen2.5-3B 推理；调用 Chart.js 生成雷达图
4. **执行输出**：经规则引擎校验，输出诊断报告与训练建议

---

## 核心亮点

- **双引擎架构**：学习计划生成 + 试卷分析集成于同一智能体
- **全本地运行**：零数据外传，绝对保护隐私
- **极简交互**：单页HTML，打开即用，核心操作≤3步
- **AI赋能开发**：从设计到demo落地仅用3小时
- **可视化输出**：雷达图直观呈现薄弱知识点，附带推理依据

---


## 部署方法

1. 安装 Ollama
2. 拉取模型：ollama pull qwen2.5:3b
3. 终端打开源代码文件所在文件夹
4. 在终端运行npm install
5. 在终端运行桥接服务：node server.js
6. 浏览器打开 index.html 即可使用

---

## 隐私说明

本智能体采用全本地部署架构，所有学习数据均在用户本机处理，不上传至任何云端服务器。
